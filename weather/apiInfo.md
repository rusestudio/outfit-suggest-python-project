# API INFO
    - The API we use is provided by the **KMA** (Korea Meteorological Administration) and makes it **easy** to retrieve weather information. 
    - And we use ather api "V world"
## API Used
    - **VilageFcsInfoService_2.0**
        - This service provides access to information on Ultra Short-Term Observations, Ultra Short-Term Forecasts, Short-Term Forecasts, and Forecast Versions.
    - **MidFcstInfoService**
        - This service provides access to information on the Medium-Range Outlook, Medium-Range Land Forecast, Medium-Range Temperature, and Medium-Range Marine Forecast.
    - **Vworld lat lon to address**
        - This service provides access to information on Korea address system
